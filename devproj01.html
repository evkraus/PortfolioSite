ARRAY DISPLAY - CODE CHALLENGE

This challenge was to display up to 6 images in a grid style using JavaScript. For this, I decided to go with the concept of potentially doing this for a photography hobby which influenced the images used. I then researched through my notes and looked at general best practices from experienced JavaScript developers online, using this information I brainstormed multiple ways of potentially accomplishing this task however there’s only one way to confirm them or not. I began playing around with the idea of using a function with the array of images I created, with this idea I was able to display the images using the “forEach” and “appendChild” statements.


let images = ["imgs/image01.jpg", "imgs/image02.jpg", "imgs/image03.jpg", "imgs/image04.jpg", "imgs/image05.jpg", "imgs/image06.jpg"];
let container = document.getElementById("imgContainer");
                    
images.forEach((imageUrl) => {
    let images = document.createElement("img");
    images.src = imageUrl;
    container.appendChild(images);
});

However, the images were not displayed in a grid style, so I went back to the drawing board. My solution to this obstacle was to create a Div using HTML, give it styling, and call it in JavaScript allowing me to complete the challenge. Once this project was over I went back revised and decided I would like to add more to prove the skills I have learnt. For this revised version I began brainstorming ideas to add to this project, the main additions I decided to add were the “Next” and “Previous” buttons; these would display the next 6 images in the grid or return to the previous 6. The other addition I added was simple animation on the buttons such as a hover and a fade-in on the images. For the process of switching to a process that can switch to another grid set, I began by setting all images into an array called ‘images’, which contains the paths to the image files that will be displayed.


const images = [
    "imgs/image01.jpg", "imgs/image02.jpg", "imgs/image03.jpg", "imgs/image04.jpg",
    "imgs/image05.jpg", "imgs/image06.jpg", "imgs/image07.jpg", "imgs/image08.jpg",
    "imgs/image09.jpg", "imgs/image10.jpg", "imgs/image11.jpg", "imgs/image12.jpg"
];

I then identified variables and contents I would need and defined them. The first one I defined was ‘currentIndex’. This is to keep track of the index of the first image displayed in the current view. Second is a constant called ‘imagesPerPage’ which specifies the number of images to be displayed at once; which is 6. Lastly I called a constant called ‘container’, this references the image container within the HTML where images will be displayed.


let currentIndex = 0;
const imagesPerPage = 6;
const container = document.getElementById("imgContainer");

Once the variables and constants have been defined I began with my functions. The first function I created was one to display images, inside the function I first cleared the container’s current content. After that I created a loop through the original array starting from ‘currentIndex’ to ‘currentIndex + imagesPerPage’, creating an ‘img’ element for each image, then appending to the container.


function displayImages() {
    container.innerHTML = '';
    for (let i = currentIndex; i < currentIndex + imagesPerPage && i < images.length; i++) {
        const img = document.createElement("img");
        img.src = images[i];
        img.classList.add('fade-in');
        container.appendChild(img);
    }
}

The next functions created were the ‘nextImages’ and ‘prevImages’ functions. The ‘nextImages’ function increments ‘currentIndex’ by ‘imagesPerPage’ to show the next set of images. The ‘prevImages’ is the same but in reverse, it decrements ‘currentIndex’ by ‘imagesPerPage’ to show the previous grid of images.


function nextImages() {
    if (currentIndex + imagesPerPage < images.length) {
        currentIndex += imagesPerPage;
        displayImages();
    }
}

Lastly I initialize the carousel by calling the ‘displayImages’ function to display the first set of images when the page loads. I also added buttons in the HTML that called to ‘pevImages’ and ‘nextImages’ functions when clicked.


function prevImages() {
    if (currentIndex - imagesPerPage >= 0) {
        currentIndex -= imagesPerPage;
        displayImages();
    }
}

--------------------------------------------

<div class="carousel-buttons">
<button onclick="prevImages()">Previous</button>
<button onclick="nextImages()">Next</button>
</div>
